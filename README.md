> A simple wrapper around [Klarnas](http://www.klarna.com/) XMLRPC API, written in nodejs.


## Where can I find releases?
You can pull them from npm:

```json
{
  "dependencies": {
    "klarna-nodejs": "1.0.0"
  }
}
```
